About This Project

An automated web scraping and analytics platform that tracks mobile phone prices daily using AWS Step Functions orchestration. Launches EC2 spot instances for cost-effective data collection with Playwright, stores data in S3, and provides real-time analytics through Athena queries and interactive visualizations.

AWS Services Used

Amazon EC2 Spot Instances
Step Functions Orchestration
AWS Lambda Functions
Amazon Athena SQL Queries
Amazon S3 Data Storage
AWS IAM Security
